## 6. 交互与动效

### Framer Motion 动效
- Section 滚入视口：淡入上滑（fade + slide up，y: 20 → 0，opacity: 0 → 1）
- Projects 卡片：hover 时 scale(1.02) + 阴影加深
- Navbar：点击锚点平滑滚动（`scroll-behavior: smooth`）
- Hero CTA 按钮：hover 渐变色过渡

### 导航
- 固定顶部 Navbar，背景半透明（`backdrop-filter: blur(10px)`）
- 滚动时添加底部阴影
- 当前 section 高亮对应导航项
